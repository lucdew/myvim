# Luc Dew's vim configuration

Greatly inspired from [Jesse Frazelle's vim](https://github.com/jessfraz/.vim)

## Linux Setup

Run the following command:

```
$ cd ~/
$ git clone --recursive https://github.com/lucdew/myvim.git .vim
$ cd $HOME/.vim
$ make install
$ make update
```
